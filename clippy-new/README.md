# Clippy New

A modern Electron-based implementation of the classic Clippy assistant, optimized exclusively for Linux.

## Platform Support

This project is Linux-only. Other platforms are not supported.

## Getting Started

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the application in development mode.

## Building for Production

To build the application for Linux, run the following command in the `clippy-new` directory:

```bash
npm run make
```

The packaged files will be available in the `out/make` directory.
